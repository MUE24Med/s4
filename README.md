🌟 نظرة عامة

خريطة المنهج الطبي التفاعلية هو مشروع متكامل لعرض وتنظيم المواد الدراسية للفرقة الثانية - الفصل الدراسي الثالث بشكل تفاعلي وسهل الاستخدام. يدعم المشروع 4 مجموعات دراسية (A, B, C, D) ويوفر وصولاً سريعاً للمحاضرات، السكاشن، الأسئلة، والإجابات.

🛠️ كيفية التشغيل

1. 📥 التحميل

```bash
git clone https://github.com/MUE24Med/s3.git
cd s3
```

2. 🌐 التشغيل المباشر

افتح index.html في أي متصفح حديث (Chrome, Firefox, Edge, Safari).

3. 🚀 خيارات متقدمة

· تشغيل سيرفر محلي: npx serve . أو npx live-server
· التطوير: فتح DevTools (F12) للتحقق من Console
